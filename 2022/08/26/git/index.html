Git

2022/8/26

git基础命令

配置用户

git config --global user.email 'yourEmail@qq.com'
git config --global user.name 'yourName'

配置文件保存在 home 目录下的 .gitconfig 文件

基本命令

git init 初始化一个git仓库
git status 显示暂存区状态
git add 文件名 / . 单文件暂存 / 所有文件暂存（使用 .gitignore 配置不暂存的文件）
git commit -m '提交信息' 提交暂存

.gitignore 配置不暂存的文件

*.html 所有html文件不暂存
/css css文件夹不暂存
/js/config.js js文件夹下的config.js不暂存
!index.js 除了index.js外
