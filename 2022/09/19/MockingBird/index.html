MockingBird使用

Anaconda安装

到镜像站下载
Index of /anaconda/archive/ | 清华大学开源软件镜像站 | Tsinghua Open Source Mirror
一般下一步到完成

配置Anaconda源

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/ 
conda config --set show_channel_urls yes

创建环境(需要python3.8以上环境运行MockingBird)

conda create -n 环境名 python=3.8

如果创建不成功，尝试修改 %USERPROFILE%\.condarc 文件为以下配置

channels:
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/win-64/
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/win-64/
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/win-64/

ssl_verify: false # 重点
show_channel_urls: true

查看所有环境

conda info --envs

进入环境

conda activate 环境名

退出环境

deactivate 环境名

删除环境

conda remove -n 环境名 --all

详细安装参考Anaconda使用教程一（新手友好） - 知乎 (zhihu.com)

下载cuda
https://developer.nvidia.com/cuda-downloads

安装cuda

选择自定义安装，可只勾选勾选cuda，不勾选vs code

查看安装版本

nvcc --version

安装 PyTorch 本地|派托奇 (pytorch.org)

成功执行以下命令则进行下一步

import torch
torch.cuda.is_available()

下载 ffmpeg ，解压后将 bin 目录添加到环境变量
Releases · BtbN/FFmpeg-Builds (github.com)
查看版本

ffmpeg -version

下载项目

git clone git@github.com:babysor/MockingBird.git

在项目目录下执行

pip install -r requirements.txt
pip install webrtcvad-wheels

若下载速度慢可以使用以下命令后重新下载

pip install pqi
pqi use aliyun

安装过程报错 Microsoft Visual C++ 14.0 or greater is required
需要安装Microsoft C++ 生成工具 - Visual Studio

下载模型到 MockingBird\synthesizer\saved_models 目录
https://pan.baidu.com/s/1PI-hM3sn5wbeChRryX-RCQ 提取码：2021

执行 python demo_toolbox.py 启动
录制一小段录音，输入文字，即可播放
若加载模型出错
MockingBird\synthesizer\utils\symbols.py 文件修改

_characters = 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz12340!\'(),-.:;? '

MockingBird\synthesizer\hparams.py 文件修改

use_gst = False
use_ser_for_gst = False
